Surfaces Soniq's reasoning — the brand's transparency mechanism. `tone="voice"` (purple) explains; `tone="inferred"` (amber) flags a guess. Optional mono `citation`.

```jsx
<ReasoningNote tone="voice" citation="cites 3 sources · cached 2h ago">
  This curator favors hazy, mid-tempo indie and placed two similar tracks this month.
</ReasoningNote>
<ReasoningNote tone="inferred">Release date set to your usual Friday — change if different.</ReasoningNote>
```
