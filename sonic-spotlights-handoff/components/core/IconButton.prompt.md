Icon-only square button. Pass a Lucide SVG as children and an accessible `label`.

```jsx
<IconButton label="More" variant="ghost"><i data-lucide="more-horizontal"></i></IconButton>
<IconButton label="Play" variant="voice"><i data-lucide="play"></i></IconButton>
```

Variants: `ghost` (default), `voice` (purple tint), `solid`.
